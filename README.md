# Weather App

A simple weather app build as part of The Odin Project JavaScript curriculum.

Live Preview: https://paulperrin-stack.github.io/weather-app/

## Features

* Search current weather by city
* Toggle between °C and °F
* Background color changes based on weather condition
* Displays temperature, feels like, humidity, and wind speed

## Built With

* HTML / CSS / JavaScript (ES6)
* Webpack + npm
* ESLint
* Visual Crossing Weather API
